### 相关工程

###### 采用前后端分离架构，前端为独立工程。
###### dna-cloud 是后台工程，核心框架：Spring Cloud Nacos

###### 前端：Vue + Element UI
##### 网关：Nginx

##### 后端：Spring Cloud
##### 数据库：MySQL
##### 缓存：Redis
##### 队列：RabbitMQ && RocketMQ && Kafka
##### 文件处理：Fastdfs && Hadoop
##### 搜索引擎：Elasticserach
##### 配置管理：Alibaba Nacos
##### 分布式事务: Alibaba Seata
##### 限流策略: Alibaba Sentinel
